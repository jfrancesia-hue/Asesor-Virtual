# Estilo Geometrico Etnico Moderno

Esta guia guarda una direccion visual reutilizable para MiAsesor y futuros proyectos. La idea es combinar confianza profesional con calidez artesanal: patrones geometricos simples, colores tierra y contrastes profundos.

## Paleta Base

- Rojo terracota: `#C44A2D`
- Rojo intenso: `#B02E2E`
- Azul petroleo: `#1E3A5F`
- Verde jade: `#1E8A73`
- Beige arena: `#E6C9A8`
- Crema claro: `#F4E1C1`

## Adaptacion A MiAsesor

MiAsesor ya tiene una base calida con terracota, crema y verde. Para no romper el producto, este estilo debe entrar como acento, no como decoracion dominante.

- Usar `#C44A2D` y `#B02E2E` solo en detalles: lineas, rombos, separadores, indicadores o pequenos patrones.
- Usar `#1E3A5F` para dar profundidad en secciones institucionales, no para llenar toda la interfaz.
- Usar `#1E8A73` como variante mas artesanal del verde de exito.
- Usar `#E6C9A8` y `#F4E1C1` como fondos suaves, especialmente en landing, perfiles profesionales y piezas comerciales.

## Recursos Visuales

- Rombos, triangulos, lineas diagonales y flechas cortas.
- Patrones simetricos en esquinas o bordes, nunca detras de texto largo.
- Texturas muy suaves en hero, footer, tarjetas destacadas o separadores.
- Bordes y contenedores limpios para mantener sensacion SaaS.

## Reglas De Uso

- En pantallas de trabajo, usar patrones al 4-10% de opacidad.
- En landing o piezas promocionales, pueden subir al 12-18% si no compiten con fotos o texto.
- Evitar fondos completamente cargados: el patron debe acompanar, no protagonizar.
- No mezclar mas de dos motivos geometricos en la misma pantalla.
- Mantener radios moderados: 8px a 24px, segun el componente.

## Componentes Donde Funciona Bien

- Hero de landing: esquina geometrica o marco sutil sobre la imagen.
- Perfil profesional: franja lateral o sello de credenciales.
- Sidebar o navbar: icono y pequenos acentos.
- Cards destacadas: borde con patron leve en una esquina.
- Material de marketing: mayor presencia de rombos y diagonales.

## Prompt Base Para Futuros Proyectos

Crear una interfaz con estilo geometrico etnico moderno, inspirada en patrones artesanales latinoamericanos, con una paleta calida y contrastante: terracota, rojo profundo, azul petroleo, verde jade, beige arena y crema claro. Usar formas geometricas simples como rombos, triangulos, lineas diagonales y flechas, con composicion simetrica, contemporanea y premium. Mantener buena legibilidad, espacios amplios y patrones como acentos visuales, no como fondo dominante.
